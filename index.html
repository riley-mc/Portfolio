On Tuesday we started off with README practices, then jumped into the beginnings of css. We also went over the importance keep code clean and organized. From here we dove into Git Branching and Merging.

Points to remember

  • $ git push rm master -f (this command is for updating a code that had been copied from my repo. Be sure to be in the directory of the one you want updated to.)
